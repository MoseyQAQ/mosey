# Agent Instructions

This repository is Mosey's personal skill set for daily research and life workflows. Treat it as the source of truth for reusable agent behavior, not as an application repository.

Mosey's research background is computational condensed matter physics. Prefer technically precise, physically explicit, and reproducible workflows. When working on research tasks, keep units, sign conventions, coordinate choices, numerical assumptions, and file provenance visible.

Skills are organized by life/research area:

- `Daily/`: everyday writing and personal workflows.
- `Research/`: research workflows, including plotting, HPC setup, and future computational-physics utilities.

When adding or editing a skill:

- Keep `SKILL.md` compact and operational.
- Put detailed reference material in `references/`.
- Put repeatable command logic in `scripts/`.
- Avoid extra README-style documentation inside skill directories.
- Validate the skill when practical before reporting completion.
